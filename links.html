Trigon Challenge

A participant-led public testing path for frameworks, diagrams, models, theories, and systems. Static website note: uploads currently happen by email until a proper form/backend exists.

  • Open the Trigon Challenge page — start with the public rules, guide paper, and participant prompt.
  • Main rule — AI can assist, but it does not judge. The participant runs the first pass.
  • Submit a result — include the framework, test question, scenarios, raw results, ambiguities, failures, and public-sharing permission.
  • Goal — build the public test ledger without turning any result into proof too early.
